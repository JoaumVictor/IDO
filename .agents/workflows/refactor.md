description: Melhora a qualidade do código sem alterar a lógica.
---

1. Analise o código selecionado em busca de complexidade desnecessária ou nomes ruins.
2. Aplique princípios de Clean Code (DRY, SOLID).
3. Mantenha o código limpo: remova sujeiras, mas não encha de comentários JSDoc se não for solicitado.
4. Mostre o "Antes" e "Depois" para o usuário.
5. Se aprovado, aplique e sugira um commit do tipo `refactor:`.
